# swift-4-weather-app

## Requirements
1. iOS SDK 10
2. Swift 4
3. Xcode 9.2
4. Cocoapod

## Steps
1. Download the repository
2. Extract it
3. Open Xcode, open the project with \*.xcworkspace
4. Run it with simulator
